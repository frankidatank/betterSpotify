# betterSpotify

Spotify player with lyrics, made using ReactJS.
Incudes using soptify-web-api-node and spotify-web-player packages

## To Run Locally
- Clone the files and use terminal, cd into server, run
  npm install
  npm start
- cd in client, run
  npm install
  npm start
